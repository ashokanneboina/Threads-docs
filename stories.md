

# User Stories for Threads App

## 1. Authentication

* As a user, I want to register an account so that I can use the platform.
* As a user, I want to log in so that I can access my account.
* As a user, I want to log out so that my account remains secure.

---

## 2. Threads (Posts)

* As a user, I want to create a thread so that I can share my thoughts.
* As a user, I want to view threads so that I can see what others are posting.
* As a user, I want to delete my thread so that I can remove unwanted content.

---

## 3. Likes

* As a user, I want to like a thread so that I can show appreciation.
* As a user, I want to unlike a thread so that I can remove my reaction.

---

## 4. Following

* As a user, I want to follow another user so that I can see their posts.
* As a user, I want to unfollow a user so that I can stop seeing their content.
* As a user, I want to view my following list so that I know whom I follow.

---

## 5. Comments 

* As a user, I want to comment on a thread so that I can express my opinion.
* As a user, I want to reply to a comment so that I can have conversations.
* As a user, I want to view comments so that I can read discussions.

---

## 6. Basic Profile Features

* As a user, I want to view my profile so that I can see my posts and activity.
* As a user, I want to view another user’s profile so that I can explore their content.
